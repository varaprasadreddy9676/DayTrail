## 7. System Tray & Menu Bar Specification

### 7.1 Asset Design & OS Adapting (The "Template" Rule)

Never use colored or static multi-color PNGs for your system tray.

* **macOS Menu Bar:** Enforces strict monochrome scaling. Icons must be drawn as transparent vector line-art and set as a **Template Image** (`icon_as_template(true)` in Tauri v2). The OS automatically flips the icon color from solid black to solid white depending on whether the user is running dark mode, light mode, or a colorful desktop wallpaper.
* **Windows System Tray:** Uses flat geometry. Dark mode requires white outlines, light mode requires dark grey outlines (`#555555`).

### 7.2 Native Context Menu Tree Architectural Schema

When a user right-clicks your tray icon, it should instantly render the system's native window dropdown, matching system fonts and hover styles flawlessly.

```
+------------------------------------+
|  DayTrail: Tracking Active     |  <- Descriptive Disabled Label (Context Indicator)
|  --------------------------------  |
|  Show Command Center     (⌥Space)  |  <- Focus Window Target
|  Quick-Capture Note      (⌥Shift+D)|  <- Spawns context scratchpad
|  --------------------------------  |

|  --------------------------------  |
|  Settings...                       |
```

## 8. Main Window Lifecycle Overrides (Close-to-Tray)

By default, hitting the close window button (`X`) kills a web-view runtime. For a native tracking daemon, we must intercept the window close event and convert it into a background execution hide state.

In your Rust lifecycle core (`src-tauri/src/lib.rs`), implement this handler block:

```rust
use tauri::{Manager, WebviewWindowEvent};

pub fn run() {
    tauri::Builder::default()
        .setup(|app| {
            // Core Tray Setup
            let toggle = tauri::menu::MenuItemBuilder::with_id("toggle", "Show Command Center").build(app)?;
            let quit = tauri::menu::MenuItemBuilder::with_id("quit", "Quit DayTrail").build(app)?;
            let menu = tauri::menu::MenuBuilder::new(app).items(&[&toggle, &quit]).build()?;
            
            let _tray = tauri::tray::TrayIconBuilder::new()
                .icon(app.default_window_icon().unwrap().clone())
                .menu(&menu)
                .icon_as_template(true) // Crucial for macOS theme mapping
                .on_menu_event(|app, event| match event.id.as_ref() {
                    "toggle" => {
                        if let Some(window) = app.get_webview_window("main") {
                            let _ = window.show();
                            let _ = window.set_focus();
                        }
                    }
                    "quit" => {
                        app.exit(0);
                    }
                    _ => (),
                })
                .on_tray_icon_event(|tray, event| match event {
                    // Left-clicking the icon pops open the primary command window instantly
                    tauri::tray::TrayIconEvent::Click { button: tauri::tray::MouseButton::Left, .. } => {
                        let app = tray.app_handle();
                        if let Some(window) = app.get_webview_window("main") {
                            let _ = window.show();
                            let _ = window.set_focus();
                        }
                    }
                    _ => (),
                })
                .build(app)?;

            Ok(())
        })
        // INTERCEPT WINDOW CLOSE EVENTS (Preventing hard death)
        .on_window_event(|window, event| match event {
            WebviewWindowEvent::CloseRequested { api, .. } => {
                api.prevent_close(); // Block the OS from wiping the memory thread
                let _ = window.hide(); // Minimize silently into your tray icon
            }
            _ => (),
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}

```
